# blinkit-dashboard
Built a Blinkit Sales & Customer Analytics Dashboard in Power BI!

Quick commerce is one of the fastest-moving spaces in Indian retail — so I built this dashboard to practice analyzing the kind of data Blinkit deals with every day.

What's inside:

📌 KPI Summary — 25T total orders, 2,172 customers, 268 products, and a 3.34 average rating, all filterable by Month, Outlet Location, and Category.

📦 Delivery Status — Orders split evenly across On Time, Slightly Delayed, and Significantly Delayed (~26K each) — a classic ops problem worth digging into.

🏆 Top 5 Best-Selling Products — Pet Treats leads (₹9,968 | 29% of grand total), followed by Dish Soap, Toilet Cleaner, Lotion, and Cough Syrup — spanning Pet Care, Household Care, Personal Care, and Pharmacy.

📅 Sales by Month — Monthly order trends showing peaks in September, March, and August, with dips mid-year.

🚨 Customer Churn Table — Each customer tagged as Active, Warning, or High Risk based on days since last active — exactly the kind of insight a retention team acts on.

🛠️ Tools: Power BI, DAX, Power Query — slicers, line charts, pie charts, data tables with conditional formatting.

📁 Data: Synthetic Blinkit-style dataset from Kaggle. Not real Blinkit data; branding used as a case study reference only.
